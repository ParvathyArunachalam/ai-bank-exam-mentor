# 🏦 AI Bank Exam Mentor

## 📖 Overview

AI Bank Exam Mentor is an AI-powered study assistant designed to help bank exam aspirants prepare smarter.

The application will allow students to:
- 📚 Upload study materials
- 💬 Ask questions from their notes
- 📝 Generate quizzes
- 📅 Create personalized study plans
- 📊 Track learning progress
- 🎯 Identify weak topics
